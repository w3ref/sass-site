CSS со сверхвозможностями

Sass - самый зрелый, стабильный и мощный язык расширений CSS профессионального уровня в мире.

  • Текущие релизы:
  • Dart Sass 1.38.2
  • LibSass 3.6.5
  • Ruby Sass ⚰
  • Руководство по внедрению

  • Совместимость с CSS

    Sass полностью совместим со всеми версиями CSS. Мы серьезно относимся к этой совместимости, чтобы вы могли беспрепятственно использовать любые доступные библиотеки CSS.

  • Богатые возможности

    Sass может похвастаться большим количеством функций и возможностей, чем любой другой язык расширений CSS. Команда Sass Core работала бесконечно, чтобы не только идти в ногу со временем, но и оставаться впереди.

  • Зрелый

    Sass активно поддерживается в течение almost 15 years его любящей основной командой.

  • Утверждено в отрасли

    Снова и снова индустрия выбирает Sass в качестве основного языка расширений CSS.

  • Большое сообщество

    Sass активно поддерживается и развивается консорциумом нескольких технологических компаний и сотен разработчиков.

  • Фреймворки

    Существует бесконечное количество фреймворков, созданных с помощью Sass. Compass, Bourbon и Susy и это лишь некоторые из них.
